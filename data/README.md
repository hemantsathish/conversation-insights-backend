# Kaggle data

Put your downloaded **Customer Support on Twitter** dataset CSV here.

**Example:** `twitter_airline.csv` or whatever the main CSV is named (e.g. `inbound.csv`, `tweets.csv`).

The load script will look here by default, or you can pass a path:

```bash
python scripts/load_kaggle_subset.py --csv data/twitter_airline.csv --limit 5000
```
